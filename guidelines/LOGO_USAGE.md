# Pica Logo Usage Guidelines

## Logo Versions

### Primary Logo (Light Mode)
The primary Pica logo should be used in most contexts on light backgrounds. Available in:
- `assets/logos/pica-logo-primary.svg` - Full color (vector)
- `assets/logos/pica-logo-primary.png` - Full color (raster)

The primary logo features the Pica magpie with:
- Deep indigo/blue body and wings aligned with Pica Dark (#1A1A2E)
- Clean white belly for contrast
- Gradient tail feathers transitioning from indigo to cyan
- Optimized for white (#FFFFFF) and light gray (Gray 100 #F8F9FA) backgrounds

**Usage**: Use this version for light mode interfaces, print materials, and standard brand applications.

### Monochrome Versions
For single-color applications:
- `assets/logos/pica-logo-white.svg` - White version for dark backgrounds
- `assets/logos/pica-logo-black.svg` - Black version for light backgrounds

### Icon Only
For small spaces or app icons:
- `assets/logos/pica-icon.svg` - Icon mark only

### Dark Mode Version
Optimized for dark interfaces using the Magpie Palette:
- `assets/logos/pica-logo-dark-mode.png` - Full color dark mode variant

The dark mode logo features the Pica magpie with:
- Deep indigo/blue body tones aligned with Royal Indigo (#2E3192)
- Iridescent teal belly accent aligned with Iridescent Teal (#005F6B)
- Subtle cyan glow effect for depth on dark backgrounds
- Optimized for Deep Obsidian (#0D0D0D) and Midnight Charcoal (#1A1A1B) backgrounds

**Usage**: Use this version exclusively on dark mode interfaces to maintain visual consistency with the Magpie Palette color system.

## Minimum Size

To ensure legibility, the logo should not be displayed smaller than:
- **Full logo**: 100px width (digital) / 25mm (print)
- **Icon only**: 24px (digital) / 6mm (print)

## Background Usage

### Approved Backgrounds
- White or light neutral backgrounds (Gray 100) - Use primary logo
- Pica Blue background - Use white logo
- Dark backgrounds (Magpie Palette) - Use dark mode logo
- Photography (ensure sufficient contrast)

### Light Mode Specific
| Background | Logo Version |
|------------|--------------|
| White (#FFFFFF) | `pica-logo-primary.png` |
| Gray 100 (#F8F9FA) | `pica-logo-primary.png` |

### Dark Mode Specific
When using the Magpie Palette dark theme:
| Background | Logo Version |
|------------|--------------|
| Deep Obsidian (#0D0D0D) | `pica-logo-dark-mode.png` |
| Midnight Charcoal (#1A1A1B) | `pica-logo-dark-mode.png` |

### Backgrounds to Avoid
- Busy patterns that reduce legibility
- Colors that clash with brand colors
- Low-contrast situations

## Logo Don'ts

- Do not stretch or distort the logo
- Do not rotate the logo
- Do not change logo colors outside of approved versions
- Do not add effects (shadows, gradients, outlines)
- Do not place the logo on low-contrast backgrounds
- Do not crop or partially obscure the logo
- Do not rearrange logo elements
- Do not use outdated logo versions

## Co-Branding

When using the Pica logo alongside partner logos:
- Maintain equal visual weight
- Use appropriate clear space between logos
- Separate with a vertical line if needed
- Ensure both logos are clearly legible
